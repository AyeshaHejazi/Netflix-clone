Netflix Clone – Landing Page

A clean, responsive recreation of the Netflix landing page built to strengthen front-end development skills. This project focuses on replicating the visual design, layout structure, and user experience of Netflix's homepage using modern web development practices.

Note: This project is intended for educational and portfolio purposes only. It is not affiliated with or endorsed by Netflix.
